# Задача 08 — Генератор пароля

## Участник группы
Петрова (группа 2991)

## Описание решения
Программа для генерации случайных безопасных паролей с настраиваемыми параметрами.

### Возможности:
- Настройка длины пароля
- Выбор наборов символов (цифры, строчные, заглавные, спецсимволы)
- Гарантия наличия минимум одного символа из каждого выбранного набора
- Генерация нескольких паролей одновременно
- Проверка сложности существующих паролей
- Оценка пароля с визуальными индикаторами

### Алгоритм работы:
1. Сбор выбранных наборов символов
2. Проверка, что длина позволяет взять по символу из каждого набора
3. Добавление по одному случайному символу из каждого набора
4. Заполнение остатка случайными символами из всех наборов
5. Перемешивание результата для равномерного распределения
6. Оценка сложности по нескольким критериям

## Как запустить

```bash
# Базовые команды
python solution.py                    # пароль 8 символов (все наборы)
python solution.py -l 12               # пароль 12 символов
python solution.py -n 5                 # 5 паролей

# Исключение наборов
python solution.py --no-digits          # без цифр
python solution.py --no-special         # без спецсимволов
python solution.py --no-upper           # только строчные

# Специальные режимы
python solution.py --only-digits        # только цифры
python solution.py --only-letters       # только буквы

# Проверка пароля
python solution.py --check MyP@ssw0rd   # проверить сложность

# Демонстрация
python solution.py --demo               # показать примеры
