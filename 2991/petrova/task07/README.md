# Задача 07 — Подсчёт частоты символов

## Участник группы
Петрова (группа 2991)

## Описание решения
Программа подсчитывает частоту каждого символа в текстовом файле 
и выводит результаты в порядке убывания частоты.

### Возможности:
- Подсчет всех символов (включая пробелы и знаки препинания)
- Опция игнорирования регистра (--ignore-case)
- Сортировка по убыванию частоты
- Визуализация с помощью полосок #
- Дополнительная статистика

### Алгоритм работы:
1. Чтение файла
2. Применение опции игнорирования регистра (если выбрано)
3. Подсчет частоты с помощью Counter
4. Сортировка результатов
5. Вывод с форматированием и визуализацией

## Как запустить

```bash
# Обычный запуск (с учетом регистра)
python solution.py test.txt

# С игнорированием регистра
python solution.py test.txt --ignore-case

# Запуск теста
python solution.py --test
