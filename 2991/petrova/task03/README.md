# Задача 03 — Переворот слов в тексте

## Участник группы
Петрова (группа 2991)

## Описание решения
Программа читает текст из файла, удаляет лишние пробелы и переворачивает порядок слов.

### Алгоритм:
1. Чтение текста из файла
2. Удаление лишних пробелов (trim и замена нескольких пробелов на один)
3. Разбиение на слова по пробелам
4. Переворот списка слов
5. Склеивание обратно с одним пробелом

## Как запустить
```bash
python solution.py input.txt
