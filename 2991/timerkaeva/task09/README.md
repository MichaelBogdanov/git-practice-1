\# Задача 09: Сжатие строки (Run-Length Encoding)



\## Участники группы

Тимеркаева Милана



\## Описание решения

Программа реализует RLE-сжатие строки и восстановление исходной строки из сжатого формата.



\### Алгоритм сжатия (encode)

1\. Проходим по строке слева направо

2\. Считаем количество повторяющихся подряд символов

3\. Записываем символ и количество повторений

4\. Пример: `AAABBBCCDAA` → `A3B3C2D1A2`



\### Алгоритм распаковки (decode)

1\. Проходим по сжатой строке

2\. Читаем символ и следующее за ним число

3\. Повторяем символ указанное количество раз

4\. Пример: `A3B2C1` → `AAABBC`



\### Особенности реализации

\- Поддержка любых символов (буквы, цифры, пробелы, знаки)

\- Работа с многозначными числами (например, `A12` распакуется в 12 букв A)

\- Корректная обработка пустой строки
