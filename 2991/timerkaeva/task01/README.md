\## Описание решения

Программа проверяет, правильно ли расставлены скобки в строке. Поддерживаются три типа скобок:

\- Круглые: `( )`

\- Квадратные: `\[ ]`

\- Фигурные: `{ }`



\### Алгоритм работы

1\. Проходим по каждому символу строки слева направо

2\. Если встречаем открывающую скобку - помещаем её в стек

3\. Если встречаем закрывающую скобку:

&nbsp;  - Если стек пуст - возвращаем "NO" (закрывающая скобка без открывающей)

&nbsp;  - Иначе извлекаем из стека последнюю открывающую скобку и проверяем, соответствует ли она текущей закрывающей

4\. Все остальные символы игнорируем

5\. В конце проверяем стек: если он пуст - все скобки закрыты правильно, иначе - есть незакрытые скобки



\### Структура решения

\- `solution.py` - основной файл с функцией `check\_brackets()`

\- `tests/` - папка с тестами (можно добавить файлы с тестовыми данными)



python solution.py
