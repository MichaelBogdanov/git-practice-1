# Задача 3 — Переворот слов в тексте

## Участники группы
Мартынов, Румянцев

## Описание решения
Программа считывает текст из файла, разбивает его на слова с помощью метода split(), который автоматически удаляет все лишние пробелы. Затем порядок слов переворачивается, и результат выводится с одним пробелом между словами.

## Как запустить
python solution.py

Программа запросит имя файла. Файл должен находиться в той же папке, что и solution.py.
