# Задача 01 — Проверка корректности скобок

## Участники группы
Мартынов, Румянцев

## Описание решения
Использован алгоритм со стеком. При встрече открывающей скобки она добавляется в стек. При встрече закрывающей скобки проверяется соответствие с последней открывающей скобкой в стеке. Если соответствие найдено, пара удаляется. Если в конце стек пуст, все скобки закрыты корректно.

## Как запустить
python solution.py
