# Задача 05 — Римские числа (двунаправленный конвертер)

## Участники группы
Мартынов, Румянцев

## Описание решения
Программа автоматически определяет тип входных данных. Для конвертации арабских чисел в римские используется таблица значений от большего к меньшему. Для обратной конвертации программа проходит по римской строке справа налево, применяя правило вычитания для случаев типа IV или IX. Валидация римских чисел выполняется путём обратной конвертации и сравнения результата с исходной строкой.

## Как запустить
python solution.py

Программа запросит число. Можно ввести арабское (например, 1987) или римское (например, IX).
