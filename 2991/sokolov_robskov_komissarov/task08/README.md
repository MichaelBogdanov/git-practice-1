# Задача 01 - Проверка корректности скобок
## Участники группы
Соколов И.к
Робсков И.А
Комиссаров Н.А

## Описание решения
Вводится 4 значения True/False, первое значение - использовать ли числа в пароле, второе - Буквы в верхнем регистре, третье - в нижнем, четвертое - спец. символы. Длинна пароля всегда 10. Если длинна пароля в итоге будет меньше 10 символов, то в него добавятся все виды по очереди.

## Как запустить
python 8.py
