# Задача 01 - Проверка корректности скобок
## Участники группы
Соколов И.к
Робсков И.А
Комиссаров Н.А

## Описание решения
Вводится строка
Проходимся посимвольно по строке
Провяется, если закрывающая скобка, то если прошлый символ та же скобка, но закрывающая то удалить открывающую
Если не та же скобка, то уже False и выход из цикла
Если скобка не закрывающая, то если это открывающая скобка, то добавить в массив. Таким образом осществляестя игнорирование других символов.
Весь цикл обёрнут в try except IndexError, чтобы если начиналось с закрывающей скобки, ловило ошибку и тоже result = False

В конце, если массив не пустой, то result = False
В конце вывод результата

## Как запустить
python 1.py
