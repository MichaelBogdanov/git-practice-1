# Задача 10 - Проверка IP-адреса

## Участники группы
Тарасевич Т.В.

## Описание решения
Программа определяет тип введённого IP-адреса - IPv4, IPv6 или INVALID. Используется стандартный модуль Python ipaddress, который позволяет работать с IP‑адресами, выполнять их проверку и нормализацию.

**Алгоритм работы**
1. Пользователь вводит IP-адрес;
2. Функция detect_ip_type(address) пытается интерпретировать ввод как IPv4, если не получается - как IPv6;
3. В случае IPv6 программа дополнительно выводит нормализованное представление адреса;
4. Если адрес не подходит под оба формата, возвращается тип INVALID.

Основные элементы программы:
* ipaddress.IPv4Address(address) - проверяет корректность формата IPv4.
* ipaddress.IPv6Address(address) - проверяет корректность формата IPv6.
* ip.compressed - возвращает сокращённый вариант IPv6-адреса (без нулей).

## Как запустить
1. Убедитесь, что установлен Python 3;
2. Скачайте файл с кодом приложения;
3. Запустите в компиляторе для языка Python;
4. Следуйте указаниям программы:
   * Введите любой IP-адрес.
   * Программа выведет тип IP-адреса и нормализованный вариант IPv6 (если это он).
