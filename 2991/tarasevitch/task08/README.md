# Задача 08 - Генератор пароля

## Участники группы
Тарасевич Т.В.

## Описание решения
Программа создаёт случайный пароль заданной длины с возможностью исключения любых символов по выбору пользователя.

Основная идея решения:
* Формируется набор всех возможных символов - цифры, буквы в верхнем и нижнем регистре, специальные символы.
* Пользователь вводит длину пароля и символы, которые нужно исключить.
* После удаления символов вызывается функция random.sample(), выбирающая случайные символы для генерации пароля.

*Если пользователь запрашивает длину пароля, превышающую количество доступных символов, программа сообщает об ошибке.*

## Как запустить
1. Убедитесь, что установлен Python 3;
2. Скачайте файл с кодом приложения;
3. Запустите в компиляторе для языка Python;
4. Следуйте указаниям программы:
    * Введите длину пароля.
    * Укажите символы, которые нужно исключить (при необходимости).
    * Получите сгенерированный пароль.
