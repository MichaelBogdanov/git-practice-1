# Задача 05 - Римские числа (двунаправленный конвертер)

## Участники группы
Тарасевич Т.В.

## Описание решения
Программа реализует двунаправленный конвертер чисел:
* Переводит римские числа (от I до MMMCMXCIX) в арабские;
* Переводит арабские числа (от 1 до 3999) в римские.

Для проверки корректности ввода используются две функции:
* num_valid_rom(num) - проверяет, состоит ли строка только из допустимых римских символов: I, V, X, L, C, D, M.
* num_valid_int(num) - проверяет, является ли ввод числом в диапазоне от 1 до 3999.

Преобразование выполняют:
* roman_to_int(num) - поочередно анализирует символы римского числа и вычисляет итоговое значение с учётом правил вычитания (например, IV = 4, IX = 9).
* int_to_roman(num) - формирует римское число, последовательно вычитая наибольшие возможные значения и добавляя нужные символы.

**Основной цикл программы обрабатывает пользовательский ввод до тех пор, пока не завершится вручную. При вводе римского числа программа переводит его в арабское, при вводе арабского - римское. Если значение не проходит валидацию, появляется сообщение "Некорректный ввод!".**

## Как запустить
1. Убедитесь, что установлен Python 3;
2. Скачайте файл с кодом приложения;
3. Запустите в компиляторе для языка Python;
4. Следуйте указаниям программы:
    * Введите римское число, чтобы перевести его в арабское.
    * Введите арабское число, чтобы перевести его в римское.
