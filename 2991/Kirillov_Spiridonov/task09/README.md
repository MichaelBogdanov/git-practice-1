# Задача 9 - Сжатие строки (Run-Length Encoding)

## Спиридонов, Кириллов
## Описание решения
Программа выполняет RLE-сжатие строки (Run-Length Encoding) и её последующее восстановление. Пользователь вводит строку с символами вручную через консоль.

Алгоритм работы:
- Функция encode() проходит по строке и считает количество подряд идущих одинаковых символов.
- Каждый символ записывается вместе с количеством повторений (например: AAA → A3).
- Функция decode() выполняет обратную операцию — восстанавливает исходную строку по формату символ + число.
- Используется цикл for для сжатия и while для декодирования.
- Поддерживаются любые символы (буквы, цифры, пробелы, спецсимволы).
## Запуск 
В IDLE -> F5 -> программа попросит ввод строки

## Пример
**Ввод строки пользователем:**
```
AAABBBCCDAA
```
**После кодирования и декодирования**
```
A3B3C2D1A2
```
```
AAABBBCCDAA
```
