# Задача 11 - Проверка шахматного хода

## Спиридонов, Кириллов
## Описание решения
Программа позволяет проверить, может ли фигура сделать ход на шахматной доске, учитывая правила движения каждой фигуры, столкновения с другими фигурами и взятие фигур противника(только для пешек).

Белые фигуры обозначаются заглавными буквами, черные – строчными. Пустая клетка – .

Алгоритм работы:
- Функция load_board() читает файл board.txt и создаёт матрицу 8x8, где каждая клетка содержит символ фигуры или . для пустой клетки.
- Функция print_board() выводит доску с координатами a–h и 1–8 для удобства понимания доски.
- Пользователь вводит исходную и целевую клетку в шахматной нотации (например: e2 → e4).
- Программа переводит шахматные координаты в индексы массива (x и y).
- Функция valid_move() проверяет, находится ли ход внутри доски, есть ли фигура на исходной клетке и не находится ли на целевой клетке своя фигура.
- В зависимости от типа фигуры (P, N, B, R, Q, K) выполняются проверки:
  - Пешка: движение на 1 клетку вперёд, на 2 клетки с начальной позиции, взятие по диагонали.
  - Конь: Г-образный ход (2+1 клетки).
  - Ладья, Слон, Ферзь: проверка пути на препятствия с помощью функции path_clear().
  - Король: ход на одну клетку в любом направлении.
- Если ход разрешён по правилам, выводится YES, иначе NO.
## Запуск 
Загрузите файлы board.txt и task11.py.
В IDLE -> F5 -> программа попросит ввод индекса шахматной фигуры  и места для перемещения этой фигуры
## Пример
**Ввод хода конем b8:**
```
Введите ход:  b8
```
**Ввод куда ходит конь с6**
```
Введите ход:  c6
```
**Программа выводит YES так как ход возможен**
```
YES
```
