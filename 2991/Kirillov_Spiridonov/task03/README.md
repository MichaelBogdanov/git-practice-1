# Задача 3 - Переворот слов в тексте

## Спиридонов, Кириллов
## Описание решения
Программа читает текстовый файл, разбивает содержимое на слова, переворачивает их порядок и выводит результат в консоль.

Использует sys.argv для получения имени файла из командной строки.

Применяет with open() для чтения файла.

split() автоматически убирает лишние пробелы, reverse() переворачивает список, join() объединяет через пробел.
## Запуск
В IDLE -> F5 -> программа попросит ввод текстового файла например 
**precols.txt:**

**До переворота слов**
```
Проверка корректности скобок
```
**После переворота слов**
```
скобок корректности Проверка
```
