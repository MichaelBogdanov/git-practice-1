# Задача 01 — Проверка корректности скобок

## Участники группы
Зимогоров А.А
Дружинин И.А

## Описание решения
1.Присутствует 2 массива, первый служит для отслеживания открывающихся скобок, второй (ключ - значение) является коллекцией всех скобок
2.В цикле, если символ является открытой скобкой то добаляем его в первый массив, если закрытой, то проверяем первый массив на наличие соответствующей открывающей скобки
3.Если условия не прошли, то возвращается значение False иначе True
 
## Как запустить
python solution.py
