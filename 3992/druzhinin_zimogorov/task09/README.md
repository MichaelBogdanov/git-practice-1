# Задача 09 — реализовать RLE-сжатие

## Участники группы
Зимогоров А.А
Дружинин И.А

1.Eсть пустой список для результата.
2.Проходим по входной строке, начиная со второго символа, сравнивая с предыдущим.
3.Если символ совпадает с предыдущим, увеличиваем счётчик.
4.Если символ отличается, добавляем в результат предыдущий символ и его счётчик, затем сбрасываем счётчик и запоминаем новый символ.
5.После окончания цикла добавляем последнюю группу.

## Как запустить
python solution.py
