# Задача 03 — Подсчёт частоты символов

## Участники группы
Гамидов Д.В.
Гаврилов В.А

## Описание решения

1. Чтение текста - получаем строку для анализа
2. Приведение регистра - опционально переводим все в нижний регистр
3. Подсчет частоты - проходим по каждому символу и считаем количество повторений (используем словарь)
4. Сортировка - упорядочиваем пары символ-частота по убыванию частоты
5. Вывод результатов - показываем таблицу с символами и их частотой
