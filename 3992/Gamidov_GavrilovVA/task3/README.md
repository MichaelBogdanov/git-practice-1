# Задача 02 — Переворот слов в тексте

## Участники группы
Гамидов Д.В.
Гаврилов В.А

## Описание решения

1. Чтение текста - получаем исходную строку
2. Разбиение на слова - метод split() удаляет все пробелы и создает список отдельных слов
3. Переворот списка - reverse() меняет порядок слов на обратный
4. Сборка результата - join() соединяет слова через один пробел
5. Вывод - показываем полученный текст
