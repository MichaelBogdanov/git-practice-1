# Задача 01 — Проверка корректности скобок

## Участники группы
Гамидов Д.В.
Гаврилов В.А

## Описание решения

Программа проверяет правильность расстановки скобок в строке, используя стек. Она идёт по символам строки, добавляя открывающие скобки в стек, а при встрече закрывающей проверяет, совпадает ли она с последней открытой в стеке. Если всё совпадает, удаляет её из стека, если нет — считает, что скобки расставлены неправильно. В конце, если стек пуст, значит скобки расставлены правильно, иначе — нет.
