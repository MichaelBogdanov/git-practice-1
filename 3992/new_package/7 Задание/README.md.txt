Задача 07 — Подсчёт частоты символов
Участники группы
Исупов Тимофей
Сашин Никита

Описание решения
Консольное приложение на C#, которое анализирует текстовый файл и подсчитывает частоту каждого символа.

Как работает:

Запрос пути к файлу у пользователя

Проверка существования файла

Чтение всего текста из файла

Подсчет количества каждого символа с использованием Dictionary<char, int>

Вывод результатов (пропуская пробелы, табуляции, переносы строк)

Отображение общего количества символов

Технологии
C#

.NET (консольное приложение)

Visual Studio

Как запустить
Открыть проект в Visual Studio

Собрать решение (Build, после Build Solution)

Запустить программу (F5 или Ctrl+F5)
