Задача 11 — Проверка шахматного хода
Участники группы
Исупов Тимофей
Сашин Никита

Описание решения
Консольное приложение на C#, которое проверяет корректность хода шахматной фигуры. Программа считывает состояние доски из файла, запрашивает ход у пользователя и определяет, возможен ли такой ход согласно правилам шахмат.

Как работает:

Создается тестовый файл board.txt с начальной позицией
Программа считывает доску (8 строк по 8 символов)
Пользователь вводит ход в формате "e2 e4" (откуда куда)
Координаты конвертируются в индексы массива
Проверяется наличие фигуры в начальной клетке
Проверяется, не бьет ли игрок свою фигуру
Проверяется возможность хода для конкретной фигуры:
Пешка (P/p): ход вперед, первый ход на 2 клетки, взятие по диагонали
Конь (N/n): ход буквой "Г"
Слон (B/b): движение по диагонали
Ладья (R/r): движение по вертикали/горизонтали
Ферзь (Q/q): движение в любом направлении
Король (K/k): движение на 1 клетку в любую сторону
Для дальнобойных фигур проверяется отсутствие препятствий на пути
Выводится результат: VALID или INVALID

Обозначения фигур
Символ	Фигура	Цвет
K Король Белые / Черные
Q Ферзь Белые / Черные
R Ладья Белые / Черные
B Слон Белые / Черные
N Конь Белые / Черные
P Пешка Белые / Черные
. Пустая клетка

Технологии
C#

.NET (консольное приложение)
Visual Studio

Как запустить
Открыть проект в Visual Studio

Собрать решение (Build, после Build Solution)

Запустить программу (F5 или Ctrl+F5)

Ввести ход в формате "e2 e4"
