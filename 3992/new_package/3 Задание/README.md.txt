# Задача 3 — Переворот слов в тексте

## Участники группы
Исупов Тимофей  
Сашин Никита

## Описание решения
Консольное приложение на C#, которое переворачивает порядок слов в тексте. Программа читает содержимое из файла `input.txt`, разбивает текст на слова, меняет их порядок на обратный и сохраняет результат в `output.txt`. Слова остаются целыми (не переворачиваются), лишние пробелы удаляются.

Алгоритм работы:
- Проверка существования входного файла 
- Чтение текста из файла
- Разделение текста на слова по пробелам
- Переворот массива слов
- Объединение слов в строку с одним пробелом
- Запись результата в выходной файл
- Вывод сообщения об успешном выполнении

## Технологии
- C#
- .NET (консольное приложение)
- Visual Studio

## Как запустить
1. Открыть проект в Visual Studio
2. Собрать решение (Build, после Build Solution)
3. Запустить программу (Ctrl + F5 или кнопка Run)
4. Файлы `input.txt` и `output.txt` появятся в папке `bin/Debug` или `bin/Release`
