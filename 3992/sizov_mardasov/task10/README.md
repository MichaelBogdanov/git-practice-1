## Описание решения
Программа принимает строку и определяет, является ли она корректным IPv4, IPv6 или невалидным адресом.

**IPv4**: 4 числа от 0 до 255, разделённые точкой. Ведущие нули запрещены (`01.02.03.04` — невалидно).  
**IPv6**: 8 групп из 1–4 шестнадцатеричных цифр, разделённых двоеточием.  
**Бонус**: при распознавании IPv6 выводится нормализованная форма (убраны ведущие нули в каждой группе).

## Как запустить
```bash
python solution.py
```
Введите IP-адрес, программа выведет: `IPv4`, `IPv6 (нормализованный: ...)` или `INVALID`.

## Примеры
```
192.168.1.1          → IPv4
2001:0db8:0000::0001 → INVALID  (сокращённая нотация :: не поддерживается)
2001:0db8:0000:0000:0000:0000:0000:0001 → IPv6 (нормализованный: 2001:db8:0:0:0:0:0:1)
256.0.0.1            → INVALID
```
