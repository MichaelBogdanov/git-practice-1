# Задача 06 — Реализация сортировки (без встроенных)

## Участники группы
Сизов  
Мардасов

## Описание решения
Реализована **сортировка слиянием (Merge Sort)**.

Алгоритм:
1. Делим список пополам рекурсивно, пока не останутся списки по 1 элементу.
2. Сливаем пары отсортированных списков обратно: на каждом шаге берём меньший из двух текущих элементов.

Встроенные функции `sort()` и `sorted()` **не используются**.

## Сложность алгоритма
| | |
|---|---|
| Время (лучший/средний/худший) | **O(n log n)** |
| Память | **O(n)** — под временные списки при слиянии |

## Как запустить
```bash
python solution.py
```
Введите числа через пробел, например: `5 3 8 1 2`
