# Задача 05 — Римские числа (двунаправленный конвертер)

## Участники группы
Сизов  
Мардасов

## Описание решения
Реализован двунаправленный конвертер:
- **Арабское → Римское**: перебираем таблицу значений по убыванию, жадно вычитая наибольшее подходящее значение.
- **Римское → Арабское**: читаем строку справа налево; если текущий символ меньше предыдущего — вычитаем, иначе прибавляем. После перевода делаем обратную проверку, чтобы убедиться в корректности записи.

Поддерживаемый диапазон: **1–3999**.

## Как запустить
```bash
python solution.py
```
Программа запрашивает ввод интерактивно. Введите число или римскую строку.

## Примеры
```
1987 → MCMLXXXVII
IV   → 4
```
