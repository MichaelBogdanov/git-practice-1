## алгоритм работы

### проверка IPv4
разделяем адрес по точкам -должно получиться ровно 4 части - каждая часть должна быть числом от 0 до 255 и числа не могут начинаться с нуля (кроме самого 0)

### проверка IPv6
разделяем адрес по двоеточиям, количество частей должно быть от 3 до 8. каждая часть может содержать только hex-символы: 0-9, a-f, A-F. длина каждой части не больше 4 символов + пустые части допускаются

### нормализация IPv6
убираем ведущие нули в каждой группе. если группа состояла только из нулей, оставляем 0 и собираем обратно через двоеточие
