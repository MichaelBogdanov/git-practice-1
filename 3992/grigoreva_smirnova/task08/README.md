Григорьева В.А.

## алгоритм работы
1. пользователь задаёт длину пароля и выбирает наборы символов
2. из каждого выбранного набора берётся по одному символу
3. оставшаяся длина заполняется случайными символами из общего пула
4. все символы перемешиваются
5. выводится готовый пароль

## запуск
python solution.py
