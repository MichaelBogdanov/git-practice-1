# Задача 10 - Проверка IP-адреса

## Участники группы
Акулова В.С.
Рыжов К.А.

## Описание решения
Функция **is_ipv4**
- Получаем массив октетов, деля строку по точкам
- Если количество октетов не равно 4, то возвращаем false
- Если какой-либо октет пустой или содержит лишний символ двоеточия, то возвращаем false
- Проходимся циклом по всем октетам
    - Если длина октета больше единицы и он начинается с нуля ("0"), то возвращаем false
    - Если октет не попадает в диапазон 0-255, то возвращаем false
- В конце возвращаем true, если все проверки были пройдены

---

Функция **is_ipv6**
- Если строка содержит невалидные символы ('.', ":::"), то возвращаем false
- Проверяем, что строка содердит два двоеточия
    - Получаем массив частей адреса, деля строку по "::"
    - Если количество частей не равно двум, то возвращаем false
    - Иначе
        - Получаем левую и правую строки всего адреса
        - Инициализируем массив левых и правых частей, деля их по ':'
        - В двух циклах проверяем, что каждый элемент левой и правой части - это валидный хекстет (Он не пустой, содержит от 1 до 4 символов включительно и каждый символ является числом в шестнадцатеричной системе)
        - Если длина левой и правой части больше или равна 8, то возвращаем false
- Иначе
    - Получаем массив хекстетов, деля строку по двоеточиям
    - Если длина массива не равна 8, то возвращаем false
    - Если какой-либо элемент пустой, то возвращаем false
    - Цикл для каждого хекстета
        - Если хекстет не в диапазоне от 1 до 4 символов включительно, то возвращаем false
        - Если хекстет не является числом в шестнадцатеричной системе, то возвращаем false
- В конце возвращаем true, если все проверки были пройдены
