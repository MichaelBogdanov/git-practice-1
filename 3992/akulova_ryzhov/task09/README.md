# Задача 09 - Сжатие строки (RLE)

## Участники группы
Акулова В.С.
Рыжов К.А.

## Описание решения
Функция кодирования:
- Если строка пустая, возвращаем пустую строку
- Запоминаем первый символ
- Итерируемся по всей входящей строке
    - Если нынешний символ равен предыдущему
    - Иначе записываем результаты в строку в формате \<символ\>\<цифра\>
- После цикла записываем остаток строки в переменную результата
- Возвращаем результат

Функция декодирования:
- Запоминаем текущую цифру и текущий символ
- Итерируемся по строке
    - Если текущий символ это цифра, то мы записываем его в переменную текущей цифры
    - Иначе
        - Проверяем не пустой ли текущий символ
        - Парсим текущее число
        - Записываем текущий символ n раз в строку результата
        - Очищаем строку текущего числа
    - Добавляем текущий символ в переменную
- После цикла делаем тоже самое с последним блоком
- Возвращаем строку результата

## Как запустить
cargo run
