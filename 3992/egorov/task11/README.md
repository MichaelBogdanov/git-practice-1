\# Задача 11 — Проверка шахматного хода



\## Участники

Егоров В.В



\## Описание

Программа проверяет, может ли шахматная фигура выполнить ход из заданной клетки в другую согласно правилам шахмат. Учитываются все типы фигур, невозможность хода сквозь другие фигуры (кроме коня), а также невозможность взятия своей фигуры.



\### Формат входного файла

Файл содержит 8 строк по 8 символов в каждой. Обозначения фигур:

\- Заглавные буквы — белые фигуры

\- Строчные буквы — чёрные фигуры

\- K, k — король

\- Q, q — ферзь

\- R, r — ладья

\- B, b — слон

\- N, n — конь

\- P, p — пешка

\- . — пустая клетка



\### Аргументы командной строки

1\. Имя файла с доской

2\. Начальная позиция (например, e2)

3\. Конечная позиция (например, e4)



\### Запуск

```bash

python solution.py input.txt e2 e4
