# Задача 08

## Участники группы
Дёмин Б.Е.


## Описание решения

Для создания пароля используется генератор рандомных чисел, который выбирает рандомное число из таблицы ASCII для цифр и букв, или рандомный индекс для массива спец-символов.

## Как запустить

Если установлен clang 
clang solution.c -o sol 
./sol
Если установлен gcc
gcc solution.c -o sol
./sol
