# Задача 07 - подсчет частоты символов

## Участник группы
Симонова Д.А

## Описание решения
Программа читает файл и подсчитывает, сколько раз встречается каждый символ.
Результат сортируется по убыванию частоты.
Если указать `--ignore-case`, регистр букв не учитывается.
Также есть проверка существования файла

## Как запустить
python solution.py input.txt
# Без учета регистра
python solution.py input.txt --ignore-case
