# Задача 04 - удаление дубликатов с сохранением порядка

## Участник группы
Симонова Д.А

## Описание решения
Программа читает файл, разбивает на элементы по пробелам и удаляет повторяющиеся, сохраняя порядок самого первого вхождения. 
Для отслеживания уже встреченных элементов
используется множество (set)

## Как запустить
python solution.py input.txt
