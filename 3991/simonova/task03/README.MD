# Задача 3 - Переворот слов в тексте
## Участник группы 
Симонова Д.А 

## Описание решения 
Программа читает файл, разбивает текст на слова (split() сам удалит лишние пробелы), переворачивает порядок слов и соединяет их обратно через пробел. Также есть проверка наличия аргумента (файла) и его существования

## Как запустить 
python solution.py input.txt
