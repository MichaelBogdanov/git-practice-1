# Задача 01 Проверка корректности скобок
## Участники группы
Митрушина В.С
Николаева С.А

## Описание решения
Создаем стек для хранения открывающих скобок потом
создаем словарь где ключи открывающие скобки, а значения закрывающие скобки
проходим по каждому символу в исходной строке
если символ открывающая скобка, то добавляем её в стек если символ закрывающая скобка, то проверяем
если стек пуст возвращаем False 
либо извлекаем последнюю открывающую скобку и соответствует ли она закрывающей
в конце проверяем, что стек пуст 

## Как запустить
python solution.py input.txt
