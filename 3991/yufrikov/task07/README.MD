# Задача 07 — Подсчёт частоты символов

## Участники группы
Юфриков М.И.

## Описание решения
Реализована функция counterSymbols, которая принимает текст и подсчитывает частоту каждого символа.
Пробелы игнорируются. Поддерживается опция игнорирования регистра — при включении весь текст приводится к нижнему регистру перед подсчётом.
Результат сортируется по убыванию частоты с помощью сортировки пузырьком

## Как запустить
python solution.py
