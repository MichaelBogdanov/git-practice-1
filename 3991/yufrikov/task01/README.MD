# Задача 01 — Проверка корректности скобок

## Участники группы
Юфриков М.И.

## Описание решения
Реализована функция check_brackets, которая проверяет корректность расстановки скобок в строке.
Используется стек — при встрече открывающей скобки она помещается в стек, при закрывающей проверяется соответствие последней открытой.
Функция возвращает YES если скобки расставлены верно, иначе NO.

## Как запустить
python solution.py
