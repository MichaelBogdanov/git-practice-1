# Задача 09 — Сжатие строки (Run-Length Encoding)

## Участники группы
Юфриков М.И.

## Описание решения
Реализованы функции rle_encode и rle_decode для сжатия и восстановления строк методом RLE.
Кодирование проходит по строке и считает подряд идущие одинаковые символы, записывая результат в формате символ + количество.
Декодирование разбирает строку посимвольно, поддерживая многозначные числа (например, A12 → 12 символов A).
Поддерживаются любые символы кроме цифр.


## Как запустить
python solution.py
