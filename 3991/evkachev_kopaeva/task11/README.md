Цель задачи (11): проверить, может ли фигура сделать ход
Ввод: файл где записано состояние шахматной доски (8 строк по 8 символов)
Был создан для выполнения задачи файл txt: board.txt

Обозначения:
Заглавные буквы - белые, строчные - черные
K - Король (King)
Q - Ферзь (Queen)
R - Ладья (Rook)
B - Слон (Bishop)
N - Конь (Knight, так как K занято королем)
P - Пешка (Pawn)

Требования:
Учесть столкновение фигур

Вывод: VALID / INVALID
