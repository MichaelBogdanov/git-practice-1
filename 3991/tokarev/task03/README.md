# Переворот порядка слов в тексте

## Описание
Простая программа на C#, которая читает текст из файла, переворачивает порядок слов и сохраняет результат.

## Функциональность
- Читает текст из файла `input.txt`
- Сохраняет слова как цельные блоки (не переворачивает буквы внутри слов)
- Удаляет лишние пробелы в начале/конце текста
- Оставляет ровно один пробел между словами
- Выводит исходный и измененный текст в консоль
- Сохраняет результат в файл `output.txt`

## Как использовать
1. Поместите файл `input.txt` в папку с программой
2. Запустите программу
3. Исходный и измененный текст отобразятся в консоли
4. Результат сохранится в файл `output.txt`
