# Задание 9: RLE-сжатие строки

## Описание
Программа реализует алгоритм сжатия Run-Length Encoding (RLE).
Заменяет повторяющиеся символы на символ и количество повторений.

## Пример работы
Вход: AAABBBCCDAA
Выход: A3B3C2D1A2


## Как использовать

### Способ 1: Ввод строки вручную
1. Запустите программу
2. Введите строку для сжатия
3. Программа покажет результат и сохранит его в файл

### Способ 2: Чтение из файла
1. Создайте файл `input.txt` со строкой для сжатия
2. Запустите программу и введите путь к файлу
3. Результат сохранится в `compressed.txt`

## Алгоритм работы
1. Проходим по строке слева направо
2. Считаем количество повторений каждого символа
3. Записываем символ и число повторений
4. Для одиночных символов тоже добавляем число (D → D1)

## Формат вывода
- Каждый символ сопровождается числом повторений
- Примеры:
  - "A" → "A1"
  - "AAA" → "A3"
  - "AB" → "A1B1"
  - "AAABBB" → "A3B3"

## Преимущества метода
- Простая реализация
- Эффективен для строк с длинными повторами
- Обратимое сжатие (можно восстановить оригинал)
