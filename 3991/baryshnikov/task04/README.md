# Задача 04 - Удаление дубликатов, сохраняя порядок

## Участники группы

Барышников И. И.

## Описание решения

1. Читаем файл с числами
2. Создаем пустой список
3. В цикле проходимся по всем числам из файла и добавляем в пустой список тогда, когда их еще там нет
4. Записываем в этот же файл список уникальных чисел через пробел

## Как запустить

python solution.py input.txt
