# Задача 06 - Реализация сортировки (без встроенных)

## Участники группы
Барышников И. И.

## Описание решения
1. Напишем фукнцию для сортировки пузырьком
2. В функции произведем сортировку пузырьком
3. Во внешнем цикле проходим N - 1 раз
4. Во внутреннем сравниваем соседние элементы
5. в результате одного прохода самый большой элемент уходит в конец списка, и так каждый проход пока не закончатся все элементы

## Как запустить
python solution.py
