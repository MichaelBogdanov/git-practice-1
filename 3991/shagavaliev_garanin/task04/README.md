## Описание алгоритма 
Программа открывает файл text.txt, считывает его содержимое и разбивает текст на слова по пробелам. Затем с помощью dict.fromkeys() создается словарь, где ключами являются уникальные слова из файла. Оператор * распаковывает ключи словаря для вывода их через пробел. Программа выводит все уникальные слова из файла text.txt в порядке их первого появления.
