## Описание алгоритма 
Есть две функции:
1. rle_code сжимает строку: считает подряд идущие одинаковые символы и пишет их как "буква + число"
2. rle_decode разжимает обратно: берет "а3б2" и превращает в "ааабб"

Программа просит ввести две строки:
1. первую — кодирует и выводит результат
2. вторую — декодирует и выводит результат

python solution.py
