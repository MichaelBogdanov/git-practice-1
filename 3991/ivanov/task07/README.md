## Описание

Необходимо открыть указанный файл, считать его содержимое и вывести частоту каждого символа.
Должна быть опция переключения игнорирования регистра, и функция сортировки по убыванию частоты.

## Инструкция по использованию:
```python
from solution import count_freqency

file_path = 'path_to_file.txt' # Замените на путь к файлу

result = count_freqency(
    file_path,
    ignore_case=False, # Замените на True для игнорирования регистра букв
    sort_output=True # Замените на False для отключения сортировки
)

print(result)
```
