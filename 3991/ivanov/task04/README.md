## Описание

Необходимо открыть указанный файл, считать его содержимое и удалить дублирующие слова или числа, разделённые пробелом, сохраняя исходный порядок.
Результат должен выводиться функцией в виде строки.

## Инструкция по использованию:
```python
from solution import remove_duplicates

file_path = 'path_to_file.txt' # Замените на путь к файлу
result = remove_duplicates(file_path)
print(result)
```
