## Описание решения
Программа генерирует случайный пароль заданной длины с выбранными наборами символов:
- цифры
- строчные буквы
- прописные буквы
- спецсимволы

**Идея:**
Пользователь выбирает, какие наборы использовать. Если длина пароля позволяет, программа гарантирует наличие минимум одного символа из каждого выбранного набора, затем добирает оставшиеся символы из объединённого набора и перемешивает результат.

**Структуры данных:**
- списки (list) для хранения выбранных наборов и символов пароля
- строки (str) для наборов символов
