## Описание решения
Для реализации выбран алгоритм **Quick Sort** (Быстрая сортировка).

**Идея:**
Алгоритм использует стратегию «разделяй и властвуй». Мы выбираем опорный элемент (*pivot*) и распределяем остальные элементы массива в три группы: те, что меньше опорного, равные ему и те, что больше. Затем рекурсивно повторяем процесс для левой и правой групп.

**Структуры данных:**
Используются динамические массивы (списки Python) для хранения разделенных частей данных.

**Сложность:**
- **В среднем:** $O(n \log n)$ — за счет эффективного разделения массива.
- **В худшем случае:** $O(n^2)$ — если массив уже отсортирован или содержит много дубликатов (при неудачном выборе опорного элемента).
- **Память:** $O(n)$ в данной реализации из-за создания новых списков на каждом этапе рекурсии.
