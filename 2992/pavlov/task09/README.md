# Программа RLE-сжатия

**Тип проекта:** Индивидуальный 

## Описание решения

Консольное приложение для кодирования и декодирования строк методом **RLE (Run-Length Encoding)** — простейшего алгоритма сжатия данных, основанного на подсчёте повторяющихся символов.

### Основные функции

- **Кодирование строки:**  
  Преобразует последовательности одинаковых символов в пары вида `символ + количество повторений`.

- **Декодирование строки:**  
  Восстанавливает исходную строку из кодированной формы.

- **Проверка корректности ввода**  
  (при выборе действия пользователем в консоли).

### Алгоритм работы

#### Кодирование
1. Проверка, что входная строка не пуста.  
2. Инициализация счётчика и текущего символа.  
3. Подсчёт количества подряд идущих одинаковых символов.  
4. Добавление результатов в список.  
5. Объединение списка в итоговую кодированную строку.  

#### Декодирование
1. Проверка, что входная строка не пуста.  
2. Посимвольный анализ закодированной строки.  
3. Определение символа и числа его повторений.  
4. Восстановление исходной последовательности.  
5. Сборка итоговой строки.
