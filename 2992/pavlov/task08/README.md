# Генератор паролей

**Тип проекта:** Индивидуальный 

## Описание решения

Консольное приложение для генерации безопасных паролей с заданными пользователем параметрами.

### Основные функции

- Генерация паролей произвольной длины  
- Настройка наборов символов:
  - Цифры: `0-9`
  - Строчные буквы: `a-z`
  - Прописные буквы: `A-Z`
  - Специальные символы: `!@#$%^&*()_+-=[]{}|;:,.<>?`
- Гарантированное включение минимум одного символа из каждого выбранного набора  
- Валидация входных данных  
- Оценка надежности пароля  

### Алгоритм работы

1. Запрос длины пароля  
2. Выбор наборов символов через интерактивный диалог  
3. Проверка минимальной длины (должна быть не меньше количества выбранных наборов)  
4. Генерация пароля с гарантированным включением символов из каждого выбранного набора  
5. Перемешивание символов для повышения случайности  
6. Вывод пароля и оценки его надежности  

### Требования

- Python 3.6 или выше
