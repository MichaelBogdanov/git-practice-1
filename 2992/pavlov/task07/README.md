# Анализ частоты символов в файле

**Тип проекта:** Индивидуальный 

## Описание решения

Консольное приложение для анализа текста в файле и подсчёта частоты встречаемости каждого символа.  
Поддерживается возможность игнорировать регистр букв (опция `--ignore-case`).

### Основные функции

- **Подсчёт частоты символов:**  
  Определяет, сколько раз встречается каждый символ в заданном файле.  
- **Сортировка по убыванию частоты:**  
  Символы отображаются начиная с наиболее часто встречающихся.  
- **Подсчёт общей и уникальной статистики:**  
  Выводит общее количество символов и число уникальных.  
- **Игнорирование регистра:**  
  При передаче параметра `--ignore-case` все символы приводятся к нижнему регистру.  
- **Обработка ошибок:**  
  Информирует о случаях, когда файл не найден или произошла ошибка при чтении.  

### Алгоритм работы

1. Проверка аргументов командной строки.  
2. Открытие указанного файла и считывание содержимого.  
3. Подсчёт количества появлений каждого символа.  
4. (Опционально) Приведение символов к нижнему регистру при включённом флаге `--ignore-case`.  
5. Сортировка результатов по убыванию частоты.  
6. Вывод статистики в удобном табличном виде.  

### Требования

- Python 3.6 или выше
