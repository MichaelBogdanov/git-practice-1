# Задача 03 — Переворот слов в тексте

## Участники группы
Васильев А. В.
Ряхов В. Д.

## Описание решения
Программа реализует алгоритм RLE-сжатия (Run-Length Encoding) для текстовых данных.
Поддерживает кодирование и декодирование файлов. Входные данные считываются из файла,
результат сохраняется в отдельный файл.

## Как запустить
```shell
# Кодирование файла
python solution.py encode input.txt encoded.txt

# Декодирование файла
python solution.py decode encoded.txt decoded.txt
```
