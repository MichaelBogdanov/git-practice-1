# Задача 03 — Переворот слов в тексте

## Участники группы
Васильев А. В.
Ряхов В. Д.

## Описание решения
Программа переворачивает порядок слов в каждой строке, удаляя дублирующиеся пробелы
Происходит чтение переданного в аргументах файла, переворот порядка слов, и запись обратно в этот же файл в кодировке utf-8

## Как запустить
python solution.py file.txt
