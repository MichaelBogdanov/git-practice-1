## Описание решения
Алгоритм для проверки корректности скобочной последовательности `([{)]}`:
- Открывающие скобки `'([{ ' → st.append(ch)` — кладём в стек
- Закрывающие скобки `')] }'` → проверяем: стек пуст? + `st.pop() == pairs[ch]`
- Словарь пар `pairs = {')':'(', ']':'[', '}':'{'}` сопоставляет пары скобок
- Финальная проверка `return not st` — стек должен быть пуст (все открывающие закрыты)
