# Задача 02 — Подсчет слов

## Участники группы
Шкиперова Валерия Анатольевна
Шкиперова Виктория Анатольевна

## Описание решения
Программа анализирует текстовый файл: подсчитывает количество строк и слов, находит самое длинное слово.
Пунктуация заменяется на пробелы с помощью `str.translate()` и таблицы замен `str.maketrans()`.
Слова ищутся через `split()`, длина отслеживается в цикле для точного контроля. 
Проверка `os.path.exists()` предотвращает ошибки с файлами.
