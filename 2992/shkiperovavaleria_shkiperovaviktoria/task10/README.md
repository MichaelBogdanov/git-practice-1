## Описание решения
**IPv4 валидация (is_ipv4):**

- Разбиваем строку по точкам — должно быть ровно 4 части

- Каждая часть: только цифры (p.isdigit()), диапазон 0-255 (int(p))

- Запрет лидирующих нулей (p != "0" and p[0] == "0"), кроме случая "0"

**IPv6 валидация (is_ipv6):**

- Проверка :: сокращения: не более одного вхождения (ip.count("::") > 1)

- Обработка ::: разбиваем на left:right, считаем недостающие блоки (8 - (len(left_parts) + len(right_parts)))

- Разбивка на 8 блоков: автодополнение нулями при :: или проверка len(parts) == 8

- Валидация блоков: длина 1-4 символа, только hex (HEX_DIGITS = set(string.hexdigits))

- Нормализация: int(p, 16) → format(int(x), "x") убирает ведущие нули

**Логика main():** IPv4 → IPv6 → INVALID с приоритетом проверки протоколов.
