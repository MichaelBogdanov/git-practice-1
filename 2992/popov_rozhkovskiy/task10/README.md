## Описание решения  
Программа определяет, является ли введённая строка корректным **IPv4** или **IPv6** адресом.  
- **IPv4**: разбивается по точкам, проверяется 4 части, каждая — число от 0 до 255 без ведущих нулей.  
- **IPv6**: разбивается по двоеточиям, 2–8 блоков, каждый до 4 шестнадцатеричных символов, без лишних символов.  
- Если формат не подходит ни под один стандарт — выводится `INVALID`.  

**Нормализация для IPv6**: для IPv6 реализована нормализация — убираются ведущие нули в каждом блоке и приводится к нижнему регистру (например, `2001:0db8` → `2001:db8`).
