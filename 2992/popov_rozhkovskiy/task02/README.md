# Задача 02 - Подсчет слов

## Участники группы
Попов И.Д.
Рожковский М.А.

## Описание решения
Читает файл с помощью with open. Подсчитывает строки через splitlines. Находит слова регуляркой bwb. Выбирает самое длинное слово функцией max с key=len. Выводит все результаты.

## Как запустить
python 2zad.py
путь к файлу
