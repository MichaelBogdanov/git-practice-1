# Задача 5 — Римские числа (двунаправленный конвертер)

## Участники группы

## Описание решения  
Программа реализует **двунаправленный конвертер** между арабскими и римскими числами в диапазоне от 1 до 3999.  

- Если на вход подаётся **арабское число** (целое, от 1 до 3999), оно преобразуется в римское по правилам вычитания и сложения значений.  
- Если на вход подаётся **римская запись**, программа проверяет её корректность (допустимые символы, правильный порядок) и преобразует в арабское.  
- Для валидации используется обратное преобразование: если `to_roman(to_arabic(s)) != s`, строка считается **некорректной**.
