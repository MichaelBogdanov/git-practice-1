# Задача 03 — Переворот слов в тексте

## Участники группы
Васильев А. В.
Ряхов В. Д.

## Описание решения
Программа переворачивает порядок строк в файле, а в каждой строке переворачивает порядок слов.
Удаляет лишние пробелы в начале и конце строк, а также сокращает множественные пробелы между словами до одного.
Обработанный текст записывается обратно в исходный файл в кодировке UTF-8.

## Как запустить
```shell
cd 2992/vasilevav_ryahov/task3
python solution.py file.txt
```
