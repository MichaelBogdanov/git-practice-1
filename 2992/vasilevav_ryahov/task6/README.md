# Задача 06 — Реализовать один из алгоритмов сортировки (MERGE)

## Участники группы
Васильев А. В.
Ряхов В. Д.

## Описание решения
Программа реализует алгоритм сортировки слиянием (Merge Sort) для сортировки чисел.
Алгоритм рекурсивно разделяет массив на меньшие части, а затем сливает их в отсортированном порядке.

## Сложность алгоритма 
- **Временная сложность**: O(n log n)
- **Пространственная сложность**: O(n) 

## Как запустить
```shell
cd 2992/vasilevav_ryahov/task6
1. Запуск со случайным списком из 50 чисел (от 1 до 100)
python solution.py

2. Запуск с чтением чисел из файла
python solution.py file.txt

3. Запуск с числами как аргументами
python solution.py 1 5 4 3
```
