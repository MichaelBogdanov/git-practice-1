\# Задача 07 — Подсчёт частоты символов



\## Описание

Программа подсчитывает частоту каждого символа в файле.

Поддерживается режим игнорирования регистра.

Результат отсортирован по убыванию частоты.



\## Входные данные

Файл `test.txt`



\## Выходные данные

Список символов и их частот.



\## Как запустить

python solution.py
