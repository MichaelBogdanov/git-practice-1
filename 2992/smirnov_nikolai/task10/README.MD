\# Задача 10 — Проверка IP-адреса



\## Описание

Программа определяет, является ли введённая строка

корректным IPv4 или IPv6 адресом.



Для IPv6 дополнительно выполняется нормализация

(удаление лишних нулей).



\## Входные данные

IP-адрес (ввод с клавиатуры)



\## Выходные данные

IPv4 / IPv6 / INVALID



\## Как запустить

python solution.py
