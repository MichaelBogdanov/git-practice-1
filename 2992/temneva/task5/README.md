# Задача 05 — Римские числа

# Участники группы
Темнева А.Р.

# Описание решения
Использован класс RomanConverter с статическими методами для конвертации в обе стороны.

Римские числа проверяются на корректность по правилам составления. Преобразование из арабских в римские работает по принципу вычитания из числа стандартных значений.

# Как запустить
python roman_converter.py
