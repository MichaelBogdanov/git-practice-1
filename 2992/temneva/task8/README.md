# Задача 08 - Генератор пароля

# Выполнила
Темнева А.Р.

# Описание решения
Использована модульная структура с функциями для каждой задачи: генерации пароля, проверки ввода, работы с пользователем.

Пароль формируется из выбранных наборов символов с гарантией минимум одного символа из каждого набора. Проверка ввода реализована через циклы с повторными запросами при ошибках.

Можно создавать много паролей подряд без перезапуска программы.

# Как запустить
python password_generator.py
