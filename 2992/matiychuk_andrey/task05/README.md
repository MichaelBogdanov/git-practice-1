# Задача 05 — Римские числа

## Участники группы
Matiychuk Andrey

## Описание решения
Программа выполняет преобразование между арабскими и римскими числами.
Поддерживаются значения от 1 до 3999.
Для римских чисел выполняется проверка корректности записи.

## Примеры
1987 → MCMLXXXVII  
IV → 4  

## Как запустить
python solution.py
