# Задача 11 — Проверка шахматного хода

## Участники группы
Matiychuk Andrey

## Описание решения
Программа считывает состояние шахматной доски из файла.
Проверяется корректность хода фигуры с учётом столкновений.
В данной реализации рассматривается ход ладьи.

## Формат входных данных
Файл board.txt содержит 8 строк по 8 символов.
'.' обозначает пустую клетку.

## Вывод
VALID — если ход корректен  
INVALID — если ход невозможен

## Как запустить
python solution.py
