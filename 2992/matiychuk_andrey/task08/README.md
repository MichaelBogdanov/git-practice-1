# Задача 08 — Генератор пароля

## Участники группы
Matiychuk Andrey

## Описание решения
Программа генерирует случайный пароль заданной длины.
Пользователь может выбрать, какие наборы символов использовать:
цифры, строчные буквы, прописные буквы и спецсимволы.

Для каждого включённого набора в пароль обязательно добавляется
как минимум один символ (если длина позволяет), после чего
оставшиеся символы выбираются случайно из всех разрешённых наборов.
Готовый пароль перемешивается.

## Как запустить
python solution.py
