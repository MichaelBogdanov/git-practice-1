\# Задача 08 — Генератор пароля



\## Участники группы

Абрамов

Андреев



\## Описание решения

Генератор создает пароль заданной длины, используя модули `random` и `string`. 

Особенности реализации:

\- Гарантированное наличие символов: программа берет по одному случайному символу из каждого выбранного набора (цифры, буквы, спецсимволы) и добавляет их в список.

\- Остаток длины заполняется случайным образом из общей массы разрешенных символов.

\- В конце используется `random.shuffle()`, чтобы обязательные символы не находились на одних и тех же позициях в начале строки, что повышает криптостойкость.



\## Как запустить

Запустите скрипт (настройки длины и наборов символов меняются внутри кода):

python solution.py
