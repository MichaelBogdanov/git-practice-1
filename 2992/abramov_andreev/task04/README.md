# Задача 04 — Удаление дубликатов

## Участники группы

Андреев

## Описание решения
Программа считывает текстовый файл и разбивает его содержимое на отдельные элементы (слова или числа) по пробелам. 
Для сохранения исходного порядка и удаления повторов используется цикл `for` и вспомогательный список `result`. 
Логика проста: элемент добавляется в итоговый список только в том случае, если его там еще нет (`if i not in result`).

## Как запустить
1. Создайте файл `data.txt` в папке с программой.
2. Запустите скрипт:
python solution.py
