# Задача 07 — Частотный анализ символов

## Участники группы
Абрамов
Андреев

## Описание решения
Программа анализирует частоту появления каждого знака в текстовом файле. 
- Реализован подсчет через словарь (`dict`), где ключом является символ, а значением — количество его вхождений.
- Добавлена опция `ignore_case`, позволяющая объединять строчные и прописные буквы.
- Вывод данных отсортирован по убыванию частоты с помощью метода `sort()`.
- Для корректного отображения спецсимволы (пробел и перенос строки) заменяются на визуальные метки `' '` и `\n`.

## Как запустить
1. Подготовьте файл `data.txt` с любым текстом.
2. Запустите скрипт:
python solution.py
