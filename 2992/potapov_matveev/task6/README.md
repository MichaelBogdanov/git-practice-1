Описание решения
Реализована сортировка пузырьком (Bubble Sort) на C#. 
Алгоритм последовательно проходит по массиву и меняет местами соседние элементы, если они расположены в неправильном порядке. 
Проходы повторяются до полной сортировки массива. Сортировка реализована без использования встроенных функций.

Анализ сложности

Временная сложность:
Худший случай: O(n*n) - массив отсортирован в обратном порядке
Средний случай: O(n*n) - случайный порядок элементов
Лучший случай: O(n) - массив уже отсортирован

Пространственная сложность: 
O(1) - сортировка выполняется на месте (in-place)
