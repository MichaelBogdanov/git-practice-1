Описание решения
Реализована программа для проверки корректности хода шахматной фигуры на C#. 
Программа считывает состояние доски из файла board.txt и проверяет введённый пользователем ход (например, "e2 e4"). Учитываются:

- Правила хода для каждой фигуры (король, ферзь, ладья, слон, конь, пешка)
- Препятствия на пути фигуры (кроме коня)
- Невозможность взятия своей фигуры

Формат входных данных:
Файл доски: 8 строк по 8 символов

Заглавные буквы — белые фигуры
Строчные буквы — чёрные фигуры
'.' — пустая клетка
K/k — король, Q/q — ферзь, R/r — ладья, B/b — слон, N/n — конь, P/p — пешка

Пример:
rnbqkbnr
pppppppp
........
........
........
........
PPPPPPPP
RNBQKBNR

Ввод хода: строка вида "e2 e4" (начальная и конечная позиции)

Как запустить:

Создать файл board.txt с начальной позицией (уже создан тестовый)

Запустить программу через task11.ехе

Ввести ход в формате "e2 e4"
