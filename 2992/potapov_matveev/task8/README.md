Описание решения
Реализован генератор случайных паролей на C# с возможностью настройки наборов символов. 
Генератор гарантирует наличие минимум одного символа из каждого выбранного набора (если длина позволяет). 
После генерации символы перемешиваются для исключения предсказуемости.

Доступные опции:
Включение/исключение цифр (0-9)
Включение/исключение строчных букв (a-z)
Включение/исключение прописных букв (A-Z)
Включение/исключение спецсимволов (!@#$%^&*()_-+=<>?)

Как запустить:
task8.exe
