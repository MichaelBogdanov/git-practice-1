# Задача 3 — Переворот слов в тексте

## Участники группы
Лунёв Александр

## Описание решения
1. Программа считывает текст из входного файла `input.txt`.
2. Удаляются начальные и конечные пробелы, текст разбивается на слова.
3. Слова переворачиваются в обратном порядке с помощью цикла, идущего с конца массива к началу.
4. Результат объединяется через пробел и записывается в файл `output.txt`.
5. Реализована обработка отсутствия входного файла.

## Как запустить
1. Поместите входной текст в файл `input.txt` в той же папке, что и программа.
2. Выполните скрипт main.py.
3. Результат будет сохранён в файле output.txt.
