# Задача 5 — Конвертер римских и арабских чисел

## Участники группы
Лунёв Александр

## Описание решения
1. Реализован конвертер:
   - **Римские → арабские**: Проверка допустимости символов, алгоритм с учётом правила вычитания (IV = 4, IX = 9 и т.д.).
   - **Арабские → римские**: Использование предопределённых пар (значение, символ) и последовательное вычитание.
2. Диапазон поддерживаемых чисел: 1–3999 (как в классической римской системе).
3. Проверка входных данных:
   - Для римских чисел — проверка символов и корректности формата.
   - Для арабских чисел — проверка диапазона.

## Как запустить
1. Выполните скрипт main.py.
2. Программа запустит тестовые примеры и выведет результаты в консоль.
