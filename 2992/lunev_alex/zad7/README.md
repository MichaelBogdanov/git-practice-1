# Задача 7 — Подсчёт частоты символов в тексте

## Участники группы
Лунёв Александр

## Описание решения
1. Программа считывает текст из файла `input.txt`.
2. Реализована опция `ignore_case` (по умолчанию `True`), которая при необходимости приводит все символы к нижнему регистру.
3. Используется словарь для подсчёта вхождений каждого символа.
4. Частоты сортируются по убыванию с использованием алгоритма пузырьковой сортировки.
5. **Обработка ошибок**: Проверка существования входного файла.

## Как запустить
1. Поместите входной текст в файл `input.txt` в той же папке, что и программа.
2. Выполните скрипт main.py.
3. Результат будет сохранён в файле output.txt.
