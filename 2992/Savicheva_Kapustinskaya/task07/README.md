# Задача 07 — Подсчёт частоты символов
## Участники группы
Савичева А.Г.
Капустинская С.А.

## Описание решения
Программа считывает текст и подсчитывает вхождения каждого символа с помощью словаря. 
Реализована возможность выбора учета регистра. 
Вывод отсортирован по частоте символов (от самых частых к редким).

## Как запустить
python solution.py
