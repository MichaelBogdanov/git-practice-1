# Задача 06 — Реализация сортировки (Пузырек)
## Участники группы
Савичева А.Г.
Капустинская С.А.

## Описание решения
Реализован алгоритм сортировки пузырьком (Bubble Sort). Алгоритм последовательно сравнивает соседние элементы и меняет их местами, если они стоят в неправильном порядке.

## Сложность алгоритма
- В худшем и среднем случае: O(n²)
- В лучшем случае (если массив отсортирован): O(n)
- Память: O(1)

## Как запустить
python solution.py
