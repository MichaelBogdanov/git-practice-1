# Задача 10 — Проверка IP-адреса

## Участники группы
Шевякова А.И
Перлова Е.А

## Описание решения
IPv4 проверяется на 4 части с числами 0-255.  
IPv6 проверяется на 2-8 шестнадцатеричных групп через двоеточия.

## Как запустить
```bash
python main.py "192.168.1.1"
python main.py "2001:0db8:85a3:0000:0000:8a2e:0370:7334"
